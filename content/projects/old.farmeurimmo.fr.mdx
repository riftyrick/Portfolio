# Mon premier site personnel

Il est en ligne depuis environ 2018, il avait été réalisé en HTML, CSS et Javascript. C'était un site web static majoritairement. Le disign était très simple
et pas forcément très agréable à regarder. Il avait pour but de présenter mes compétences.

J'ai progressivement ajouté des pages et du responsive mais le vrai changement est arrivé avec la v2.

Il est toujours disponible à [cette adresse](https://farmeurimmo.github.io/old.farmeurimmo.fr/index.html).

Il n'est plus maintenu depuis 2021-2022. La repo github a été supprimé et recréé à plusieurs reprise car j'avais décidé de passer
sous [bootstrap](https://getbootstrap.com/).

La repo github est disponible à [cette adresse](https://github.com/Farmeurimmo/old.farmeurimmo.fr/tree/v1).

# Mon deuxième site personnel

Il est en ligne depuis environ 2022, il avait été réalisé avec le framework React. Il est toujours en ligne à [cette adresse](https://v2.farmeurimmo.fr).

Celui ci était beaucoup plus complet et mieux réalisé. Il avait pour but de présenter mes projets, mes compétences et de découvrir de nouvelles technologies.

Il n'est plus maintenu depuis Juillet 2023.

J'avais essayé je dis bien essayé de faire un formulaire de contact mais, malgré mes nombreuses tentatives, je ne suis pas arrivé à le faire fonctionner sans montrer des logins api.
De plus, je n'avais ni la motivation ni la machine pour faire une petite api pour traiter mes messages. J'ai donc décidé de l'enlever mais je songe actuellement faire une petite api et application mobile pour le faire.

La repo github du site est disponible à [cette adresse](https://github.com/Farmeurimmo/old.farmeurimmo.fr/tree/v2).
